# Chapter 1: Introduction

This chapter introduces the fundamental concepts.

## 1.1 Motivation

The core idea is simple: convert readable Markdown into typeset-quality LaTeX
automatically, handling the most common structural elements without manual
intervention.

## 1.2 Block Quotes

A tab-indented block quote (one tab or four leading spaces):

	This is a block quote rendered from a tab-indented line.
	It can span multiple consecutive tab-indented lines.

A standard Markdown block quote uses the `>` prefix:

> Albert Einstein once said that imagination is more important than knowledge.
> Knowledge is limited; imagination encircles the world.
